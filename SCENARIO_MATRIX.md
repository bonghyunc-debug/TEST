# Smart Capital Gain Wizard - 시나리오 매트릭스

## 개요

이 문서는 양도소득세 간편신고서 작성 프로그램에서 구현 가능한 모든 시나리오를 정의합니다.
각 시나리오별로 필요한 입력 필드, 적용되는 계산 로직, 출력 결과를 명확히 규정합니다.

**핵심 원칙**: 상황별 자동 분기로 필요한 입력만 받아 진행

---

## 1. 분기 축 (Branching Axes)

### 1.1 1차 분기: 신고유형

| 코드 | 신고유형 | 설명 |
|-----|---------|------|
| `regular` | 예정신고 | 양도일로부터 2개월 이내 정상 신고 |
| `after_deadline` | 기한후신고 | 신고기한 경과 후 자진 신고 |
| `amended` | 수정신고 | 기신고 내용 수정/추가 신고 |

### 1.2 2차 분기: 자산유형

| 코드 | 자산유형 | 세율 특성 | 장특공 |
|-----|---------|----------|-------|
| `general_house` | 일반주택 | 기본/단기중과 | 일반 |
| `high_price_house` | 1세대1주택(고가) | 기본/단기중과 | 1세대1주택 |
| `commercial` | 상가/건물 | 기본/단기중과 | 일반 |
| `land` | 토지 | 기본/비사업용중과/단기중과 | 일반 |
| `land_farm` | 농지 | 기본/비사업용중과/단기중과 | 일반 |
| `presale_right` | 분양권 | 기본/단기중과(강화) | 배제 |
| `membership_right` | 조합원입주권 | 기본/단기중과(강화) | 배제 |
| `unregistered` | 미등기 | 70% 단일 | 배제 |

### 1.3 3차 분기: 취득원인

| 코드 | 취득원인 | 취득가액 방법 | 보유기간 기산 |
|-----|---------|-------------|-------------|
| `purchase` | 매매 | 실지/환산 | 취득일 |
| `construction` | 신축 | 실지/환산 | 취득일 |
| `auction` | 경매/공매 | 실지/환산 | 취득일 |
| `inheritance` | 상속 | 실지/기준시가 | 피상속인 취득일 |
| `gift` | 증여 | 실지/기준시가 | 증여일 |
| `gift_carryover` | 증여(이월과세) | 당초 취득원인 따름 | 당초 취득일 |

### 1.4 4차 분기: 취득일자 구간

| 구간 | 기준일 | 적용 로직 |
|-----|-------|----------|
| A | ~ 1984.12.31 | 의제취득일 1985.1.1 적용 |
| B | 1985.1.1 ~ 1990.8.29 | 토지등급 환산 (토지만) |
| C | 1990.8.30 ~ | 현행 방식 |

### 1.5 5차 분기: 양도원인

| 코드 | 양도원인 | 특이사항 |
|-----|---------|---------|
| `sale` | 매매 | 일반 |
| `expropriation` | 수용 | 감면 가능 |
| `auction` | 경매/공매 | 일반 |
| `burden_gift` | 부담부증여 | 채무비율 안분, 신고기한 3개월 |
| `exchange` | 교환 | 일반 |

## 2. 시나리오 카탈로그

### 2.1 자산유형별 기본 시나리오

#### SC-001: 일반주택 양도

```
분기 조건:
  - 자산유형: general_house
  - 취득원인: purchase, construction, auction
  - 양도원인: sale, auction, exchange

입력 필드:
  [필수] 양도일, 취득일, 양도가액, 취득가액(실지/환산), 필요경비
  [선택] -

적용 로직:
  - 세율: 보유기간별 (2년 이상 기본세율, 1~2년 40%, 1년 미만 50%)
         ※ 조정대상지역 다주택자는 별도 중과 (본 프로그램 미지원)
  - 장특공: 일반 (3년 이상 보유, 연 2%, 최대 30%)
  - 가산세: 신고유형에 따름
```

#### SC-002: 1세대1주택 고가주택 양도

```
분기 조건:
  - 자산유형: high_price_house
  - 양도가액: 12억 초과

입력 필드:
  [필수] 양도일, 취득일, 양도가액, 취득가액, 필요경비, 거주기간
  [선택] -

적용 로직:
  - 과세대상: 12억 초과분만 과세 (비과세 안분)
  - 세율: 기본 누진세율 (비과세 충족 시 단기중과 미적용)
  - 장특공: 1세대1주택 (보유 연4% + 거주 연4%, 최대 80%)
           ※ 거주 2년 미만 시 일반 장특공 적용
```

#### SC-003: 상가/건물 양도

```
분기 조건:
  - 자산유형: commercial

입력 필드:
  [필수] 양도일, 취득일, 양도가액, 취득가액, 필요경비
  [선택] -

적용 로직:
  - 세율: 기본 누진세율 / 단기중과(1년 미만 50%, 1~2년 40%)
  - 장특공: 일반 (3년 이상, 연 2%, 최대 30%)
```

#### SC-004: 토지 양도 (사업용)

```
분기 조건:
  - 자산유형: land
  - 토지용도: business (사용자 선택)

입력 필드:
  [필수] 양도일, 취득일, 양도가액, 취득가액, 필요경비, 면적
  [조건부] 토지등급 정보 (1990.8.30 이전 취득 시)
  [선택] -

적용 로직:
  - 세율: 기본 누진세율 / 단기중과(1년 미만 50%, 1~2년 40%)
  - 장특공: 일반 (3년 이상, 연 2%, 최대 30%)
```

#### SC-005: 토지 양도 (비사업용)

```
분기 조건:
  - 자산유형: land
  - 토지용도: non_business (사용자 선택)

입력 필드:
  [필수] 양도일, 취득일, 양도가액, 취득가액, 필요경비, 면적
  [조건부] 토지등급 정보 (1990.8.30 이전 취득 시)
  [선택] -

적용 로직:
  - 세율: 기본세율 + 10%p (비사업용 토지 중과)
         ※ 단기중과와 비교과세 (더 높은 세액 적용)
  - 장특공: 일반 (3년 이상, 연 2%, 최대 30%)
```

#### SC-006: 농지 양도

```
분기 조건:
  - 자산유형: land_farm

입력 필드:
  [필수] 양도일, 취득일, 양도가액, 취득가액, 필요경비, 면적
  [필수] 사업용/비사업용 선택
  [조건부] 토지등급 정보 (1990.8.30 이전 취득 시)
  [선택] 감면 유형 (8년 자경 등)

적용 로직:
  - 세율: 사업용 - 기본세율, 비사업용 - 기본세율+10%p
  - 장특공: 일반 (3년 이상, 연 2%, 최대 30%)
  - 감면: 8년 자경 100% (한도 1억/5년), 대토 100% 등
```

#### SC-007: 분양권 양도

```
분기 조건:
  - 자산유형: presale_right

입력 필드:
  [필수] 양도일, 취득일, 양도가액, 취득가액, 필요경비
  [선택] -

적용 로직:
  - 세율: 1년 미만 70%, 1~2년 60%, 2년 이상 기본세율
         ※ 주택 분양권 강화된 단기중과
  - 장특공: 배제 (적용 불가)
```

#### SC-008: 조합원입주권 양도

```
분기 조건:
  - 자산유형: membership_right

입력 필드:
  [필수] 양도일, 취득일, 양도가액, 취득가액, 필요경비

적용 로직:
  - 세율: 1년 미만 70%, 1~2년 60%, 2년 이상 기본세율
```

#### SC-009: 미등기 자산 양도

```
입력 필드:
```

#### SC-010: 상속 취득 자산 양도

```
분기 조건:
  - 취득원인: inheritance
  - 자산유형: 모든 유형

추가 입력 필드:
  [필수] 피상속인 취득일 (세율/장특공 기산일)

적용 로직:
```

```
  - 자산유형: 모든 유형

추가 입력 필드:
  [필수] 당초 증여자 취득일
  [필수] 당초 증여자 취득원인 (매매/상속/증여)

  [필수] 취득가액 계산방법: 당초 취득원인에 따라 결정
         - 당초 매매: 실지취득가액 또는 환산취득가액
         - 당초 상속/증여: 실지취득가액 또는 기준시가

  - 보유기간: 당초 증여자 취득일 ~ 양도일
  - 취득가액: 당초 증여자의 취득가액 승계
  - 세율/장특공: 당초 취득일 기준 보유기간 적용
  - 기납부 증여세: 양도소득세에서 공제
```

### 2.3 취득일자별 시나리오

#### SC-020: 1985.1.1 이전 취득 자산

```
분기 조건:
  - 취득일: ~ 1984.12.31
  - 자산유형: 모든 유형

적용 로직:
  - 의제취득일: 1985.1.1 적용
  - 보유기간: 1985.1.1 ~ 양도일
  - 장특공/세율: 의제취득일 기준 계산

비고:
  - 실제 취득일이 아무리 오래되어도 1985.1.1이 기산점
  - 토지인 경우 토지등급 환산도 함께 적용
```

#### SC-021: 1990.8.30 이전 취득 토지 (매매 취득)

```
분기 조건:
  - 자산유형: land, land_farm
  - 취득원인: purchase, construction, auction
  - 취득일: 1985.1.1 ~ 1990.8.29

추가 입력 필드:
  [필수] 면적 (㎡)
  [필수] 1990.1.1 개별공시지가 (원/㎡)
  [필수] 취득 시 토지등급 (1~365)
  [필수] 1990.8.30 토지등급 (1~365)
  [필수] 1990.8.30 직전 토지등급 (1~365)

적용 로직:
  - 취득당시 기준시가 = 토지등급 환산 계산
    공식: (1990.1.1 공시지가 × 취득시등급가액) / ((90.8.30등급가액 + 직전등급가액) / 2) × 면적
  
  - 취득가액 계산방법: 환산취득가액
    공식: 양도가액 × (취득당시 기준시가 / 양도당시 기준시가)
```

#### SC-022: 1990.8.30 이전 취득 토지 (상속/증여 취득)

```
분기 조건:
  - 자산유형: land, land_farm
  - 취득원인: inheritance, gift, gift_carryover(최초 상속/증여)
  - 취득일: 1985.1.1 ~ 1990.8.29

추가 입력 필드:
  [필수] 면적 (㎡)
  [필수] 1990.1.1 개별공시지가 (원/㎡)
  [필수] 취득 시 토지등급 (1~365)
  [필수] 1990.8.30 토지등급 (1~365)
  [필수] 1990.8.30 직전 토지등급 (1~365)

적용 로직:
  - 취득당시 기준시가 = 토지등급 환산 계산 (SC-021과 동일)
  
  - 취득가액 계산방법: 기준시가
    취득가액 = 토지등급 환산으로 계산된 취득당시 기준시가
    ※ 환산취득가액 방식 사용 불가 (실거래가 없음)
```

### 2.4 양도원인별 시나리오

#### SC-030: 부담부증여

```
분기 조건:
  - 양도원인: burden_gift
  - 자산유형: 모든 유형

추가 입력 필드:
  [필수] 증여재산 평가방법: 시가 또는 보충적평가(기준시가)
  [필수] 증여재산 평가액
  [필수] 채무인수액

적용 로직:
  - 양도비율 = 채무인수액 / 증여재산평가액
  - 과세대상 = 양도가액 × 양도비율
  - 취득가액/필요경비: 양도비율로 안분
  - 신고기한: 증여일로부터 3개월

취득가액 계산방법 분기:
  1. 증여재산 평가방법 = 시가:
     → 취득원인에 따른 기존 분기 로직 적용
     - 매매/신축/경매: 실지취득가액 또는 환산취득가액
     - 상속/증여: 실지취득가액 또는 기준시가
     - 이월과세: 당초 취득원인에 따름
  
  2. 증여재산 평가방법 = 보충적평가(기준시가):
     → 취득가액 계산방법 = 기준시가 (강제 적용, 선택 불가)

비고:
  - 양도소득세: 채무인수액 비율만큼
  - 증여세: 나머지 비율 (별도 신고)
```

#### SC-031: 공익사업 수용

```
분기 조건:
  - 양도원인: expropriation
  - 자산유형: 모든 유형

추가 입력 필드:
  [선택] 감면 유형 (현금보상 / 대토보상)

적용 로직:
  - 현금보상: 양도소득세 10% 감면 (3년 이상 보유, 한도 1억)
              양도소득세 15% 감면 (5년 이상 보유, 한도 2억)
  - 대토보상: 양도소득세 과세이연 가능
```

### 2.5 신고유형별 시나리오

#### SC-040: 예정신고

```
분기 조건:
  - 신고유형: regular
  - 신고일: 법정기한 이내

입력 필드:
  [필수] 양도일
  [필수] 신고일
  [필수] 납부예정일

적용 로직:
  - 가산세: 없음
  - 신고기한: 양도일 말일로부터 2개월
            (부담부증여는 3개월)
```

#### SC-041: 기한후신고

```
분기 조건:
  - 신고유형: after_deadline
  - 신고일: 법정기한 경과

입력 필드:
  [필수] 양도일
  [필수] 신고일
  [필수] 납부예정일

적용 로직:
  - 무신고 가산세: 납부세액 × 20% (부정 40%)
    감면: 1개월 이내 50%, 1~3개월 30%, 3~6개월 20%
  
  - 납부지연 가산세: 미납세액 × 지연일수 × 0.022%
    (법정기한 다음날 ~ 납부일)
```

#### SC-042: 수정신고

```
분기 조건:
  - 신고유형: amended
  - 기신고 내역 존재

입력 필드:
  [필수] 양도일
  [필수] 신고일
  [필수] 납부예정일
  [필수] 당초 납부 양도소득세
  [필수] 당초 납부 농어촌특별세

적용 로직:
  - 과소신고 가산세: 추가납부세액 × 10% (부정 40%)
    감면: 1개월 90%, 1~3개월 75%, 3~6개월 50%,
         6개월~1년 30%, 1~1.5년 20%, 1.5~2년 10%
  
  - 납부지연 가산세: 추가납부세액 × 지연일수 × 0.022%
    (당초 법정기한 다음날 ~ 납부일)
```
